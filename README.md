# ✏️ TextEDitz

TextEDitz is a simple and fast text editor built using React. It helps users quickly format, analyze, and manipulate text in a clean and responsive interface.

---

## 🚀 Features

- Convert text to UPPERCASE and lowercase
- Word and character count
- Remove extra spaces
- Copy text to clipboard
- Live text preview

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

---

## 📦 Installation

Clone the repository:
  
  "git clone https://github.com/Msaad747/TextEditz.git"

Navigate to project folder:

  cd texteditz

Install dependencies:

  npm install

Run the app:

  npm start
